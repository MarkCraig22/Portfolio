Crossfit Games

Data from the CrossFit Games 2019, featuring two interactive dashboards. The first dashboard focuses on selected athlete information and results, allowing users to filter and view detailed performance data, trends, and event outcomes. The second dashboard provides performance statistics by country and affiliated gym, utilizing bar charts to offer insights into gym affiliations of top-performing athletes. This project showcases my ability to leverage Excel for data analysis and visualization, enabling users to explore and derive insights from complex data sets.

Nashville Housing

This SQL project focuses on cleaning a Nashville housing dataset, involving tasks such as standardizing date formats, populating missing property addresses, splitting address and owner address fields, identifying and removing duplicates, and dropping unnecessary columns. The project demonstrates proficiency in data cleaning techniques, SQL queries, and string manipulation, preparing the dataset for in-depth analysis, reporting, model building, and integration with other datasets in real estate-related domains.
